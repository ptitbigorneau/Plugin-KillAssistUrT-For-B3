# killassisturt plugin
# Plugin for B3 (www.bigbrotherbot.com)
# www.ptitbigorneau.fr

killassisturt B3 plugin (v1.1.2) for Urban Terror

Installation:

1. Place the killassisturt.py in your ../b3/extplugins and the 
killassisturt.ini in your ../b3/extplugins/conf folders.

2. Open your B3.xml file (default in b3/conf) and add the next line in the
<plugins> section of the file:

<plugin name="killassisturt" config="@b3/extplugins/conf/killassisturt.ini"/>
